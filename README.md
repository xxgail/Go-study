## 1. 跟PHP的比较
 
 1. PHP中的数组，在Go中分为两种
    ```
    //php
        $arr1 = [1,2,3];
    //Go
        arr1 := [3]int{1,2,3}
        
    //php
        $arr2 = [
            'a' => 1,
            'b' => 2,
        ]
        
    //Go
        arr2 := map[string]int{"a":1,"b":2}       
       
    ```
